# 次世代Pokemon傷害計算器
有別於傳統單對單計算器，為單對多的傷害計算器  
[https://catho7.github.io/NextGenPokemonDamageCalc/](https://catho7.github.io/NextGenPokemonDamageCalc/)
## 未實裝功能/內容
* 部份圖片
* 部份招式
* 部份怪物的其他形態資料
* 道具效果
* 特性效果
* 招式效果
* 天氣、場地
* 複數對象時的傷害計算
* 急所傷害計算
* etc
